
Стартовый шаблон для верстки адоптивных шабонов HTML5&CSS3



Стартовый шаблон позволяет быстро начать верстать Landing pages в его состав включены:

bootstrap сетка
самые востребованые плагины и дополнения
каркас из header, кнопки вызова всплывающего окна с формой обратной связи, слайдера отзывов
 (команды, брендов)
 подключены форма отправки сообщений на почтовый ящик.

В качестве фона header используется изобращение width = 1920рх

Верстка с помощью препроцессора scss, сборщик - prepros.



В каталоге scss размещены следующие файлы:

var.scss - размещены переменные и миксины
main.scss - файл в который интегрированы все файлы scss, в него вставлять пользовательские стили
оформления разметки не рекомендовано, можно инклудить стили с остальных файлов *.scss
media.scss - для придания полной адаптивности шаблона
header.scss - стили оформления header шаблона, в нем уже реализовано оформление шапки
по умолчанию. Это сделано для сокращения времени верстки шапки сайта, так как они в Landing Pages
имеют обычную структуру и наполнение.
fonts.scss - выполняется привязка индивидуальных шрифтов
allstyle.scss - прописываются все стили блоков оформления шаблона, которые идут по разметке
ниже header
formstyle.scss - оформление всех размещенных форм на Landing Pages. Я их выделил в обдельный файл
так как оформдение и контролль наз внешним видом ФОС довольно трудомки и требует использования
большого количества стилей.

Весь код минимизирован и оптимизирован.

Пользовательские стили собраны в файл main.css. Их необходимо минимизировать и вставить в файл libs.css

Удалите строку из файла index.html, которая привязывает файл main.css

Блок подключения скриптов

К стартовому шаблону привязаны такие script-плагины

jquery-2.1.3.js
owl carousel.js
equalHeights.js
nicescroll.js
magnific popup.js
PageScroll2id.js
common.js

Во время сборки продакшт версии необходимо все содержимое файлов собрать последовательно в один js файл, к примеру, all-common.js, код минимизирован, поэтому это не нужно делать самостоятельно.

При необходимости подключите другие плагины к стартовому шаблону для этого уже созданы файлы libs.css и jquery-2.1.3.js.

Во время сборки были к стартовому шаблону включены только самые популярные и необходимые компоненты, которые могут потребоваться для верстки простых и средних по сложности Landing Pages

К вниманию пользователей!!!
